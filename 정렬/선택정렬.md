## 정렬

### 선택정렬

- 선택해서 교환한다. 

- 실제 코테에서는 느려서 잘 쓰진 않는다. 

예시) 작은 순으로 정렬하기

```python
data = [3, 1, 4, 5, 2]

for i in range(len(data)):
    min_index = i
    for j in range(i, len(data)):
        if data[min_index] > data[j]:
            min_index = j
    data[i], data[min_index] = data[min_index], data[i]

print(data)
```

**시간복잡도**

(N-1) + (N-2) ... 2 ⇒ N * (N-1) / 2 ⇒ O(N^2)
